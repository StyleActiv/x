# 🛍️ Estilo Activo – Tienda Web con Carrito y Pedidos vía WhatsApp

**Estilo Activo** es una tienda web moderna y responsive, pensada para pequeños negocios y emprendedores que desean vender productos físicos de forma rápida, sencilla y sin complicaciones técnicas.

El sitio incluye un catálogo visual, carrito de compras funcional, y pedidos automatizados por **WhatsApp**, eliminando la necesidad de pasarelas de pago o backends complejos.

## 🧰 Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (vanilla)**
- **GitHub Pages** (para hosting gratuito)
- **WhatsApp API** (para enviar pedidos)

---

## ✅ Características principales

- 🛒 **Carrito de compras dinámico**: agrega múltiples productos, cantidades y totaliza el pedido.
- 📲 **Checkout por WhatsApp**: genera automáticamente el mensaje con los productos seleccionados y lo envía al número del negocio.
- 🎨 **Diseño adaptable (responsive)**: compatible con dispositivos móviles y escritorio.
- 🧩 **Categorías prediseñadas**: camisetas, tenis, jeans, cascos, deportes, regalos (totalmente personalizables).
- 💬 **Formulario de contacto**: para recibir mensajes desde la web.
- 📄 **Secciones legales incluidas**: términos y condiciones, política de privacidad.
